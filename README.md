# 上下ルビ組版ツール

ブラウザ上で動作する、動画や画像作成のための高度なテキスト組版（ルビ振り、縦書き翻訳文など）を行う専用ツールです。

## 主な機能

*   **3段テキスト組版**: 「本文」「上ルビ（漢字など）」「下ルビ（カナ）」を綺麗に揃えて配置します。
*   **ルビの自動同期**: `{本文|上ルビ}` という独自の構文を使うことで、文字数に応じたルビの均等割り付けを自動で計算・描画します。上ルビ内の `()` を使った特殊な文字数調整にも対応しています。
*   **特殊カナ対応**: 下ルビに入力されたアイヌ語の小書きヌ「ㇴ」を、自動的に小さな「ン」に変換して綺麗に描画します（チェックボックスでON/OFFの切り替えが可能）。
*   **翻訳文の縦書き配置**: 画面右側に縦書きの翻訳テキストを独立して配置可能です。
*   **柔軟な書式設定**: 各テキスト要素（本文・上ルビ・下ルビ・翻訳文）ごとに、フォント、太さ、サイズ、色、縁取りを個別に細かく設定できます。
*   **PCフォントの読み込み**: 「PCフォント読込」ボタンを押すことで、お使いのPCにインストールされているすべてのローカルフォントを読み込み、プルダウンから選択できるようになります（※Chrome/Edgeなどの対応ブラウザのみ）。
*   **設定の自動保存とプリセット**: 編集中の内容は自動的にブラウザ（LocalStorage）に保存され、次回開いたときに復元されます。また、現在の設定を `.json` ファイルとして「保存」したり、「読込」して設定を切り替えることができます。
*   **高解像度PNG出力**: 作成した組版を、指定したキャンバスサイズ（フルHDなど）の高解像度PNG画像としてワンクリックで書き出せます。

## 使い方

1.  **起動**: このフォルダにある `index.html` を、Google Chrome または Microsoft Edge で開きます。
2.  **テキストの入力**:
    *   本文には `{전화|戰火}` のように、波括弧と縦線を使って本文と上ルビを入力します。
    *   下ルビには、本文の改行位置と合わせてカナを入力します。
    *   翻訳テキストには、日本語の翻訳などを入力します。
3.  **デザインの調整**:
    *   右側のパネルでフォントや色、キャンバスサイズを調整します。
    *   プレビュー画面（左側）にリアルタイムで反映されます。
4.  **書き出し**:
    *   完成したら、右上の「PNG出力」ボタンを押すと画像がダウンロードされます。

## ファイル構成

*   `index.html`: アプリケーションのUI構造を定義するファイル。
*   `style.css`: UIのプロ仕様デザイン（ライトモード）および、プレビュー画像のスタイル・配置を定義するファイル。
*   `script.js`: アプリケーションのすべての動作（文字の解析・描画、フォント読み込み、設定の保存、画像化など）を制御するJavaScriptファイル。メンテナンス性を高めるためクラス化（オブジェクト指向）で整理されています。

## 注意事項

*   PCフォントの読み込み機能は、セキュリティの都合上、ブラウザを開いた直後には自動で実行されません。使用する際は必ず手動で「PCフォント読込」ボタンをクリックして許可を与えてください。
*   ローカルファイルの画像を扱うため、画像書き出し（`html2canvas`）には一部ブラウザで制約が出ることがあります。Chrome環境での利用を推奨します。
